## 2. スコープ

### 含めること

- 対象地域、施設タイプ、利用場面の未確定範囲を明示する。
- 子どもの遊び場、買い物体験、地域接点に関する代替手段を整理する。
- 代替手段と本事業の差分を、競合優劣ではなく位置づけ仮説として整理する。
- 参考事例から学べることと、そのまま適用してはいけないことを分ける。
- 利用者ニーズを事実ではなく仮説として扱い、検証方法を付ける。

### 含めないこと

- 市場規模、対象地域の需要、利用者ニーズの断定。
- 特定施設、自治体、地域団体の承認済み表現。
- 子ども食堂、学童、地域イベントなどの参考事例を成功根拠として扱うこと。
- 食品衛生、営業許可・届出、保険、責任範囲の要否判断。
- 外部説明文、営業資料、施設向け確定文の作成。

## 11. BP-02 / BP-04 / BP-05へ統合できる素材案

以下は統合素材であり、確定本文ではない。

> この事業は、既存の遊び場、放課後の居場所、地域イベント、商店での買い物を置き換えるものではない。現時点では、駄菓子を媒介にして、子どもが小額の範囲で自分で選び、金額を意識し、周囲とやり取りする小さな体験を検証する位置づけとする。
>
> 放課後児童クラブや児童館などは、子どもの居場所や遊びの場として重要な隣接環境である。一方、本事業は保育や生活支援制度ではなく、買い物体験、選択、会話、地域接点を小さく観察する仮説事業として整理する。子ども食堂や地域食堂は、地域の子どもと食品を扱う活動として衛生管理や安全説明の重要性を参照するが、本事業の需要や成功可能性を示す根拠としては扱わない。
>
> 利用者ニーズは未検証である。初回実証では、子どもが選ぶ体験に関心を示すか、保護者や施設が安全・衛生・負担面を許容できるか、既存の代替手段では満たしにくい体験価値があるかを、観察とヒアリングで確認する。
