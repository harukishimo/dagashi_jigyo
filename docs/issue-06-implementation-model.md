### 含めること

- 移動式、施設内、イベント出店などの実施形態候補を比較する。
- 初回実証に向く場所条件、頻度、時間帯、規模を仮説として整理する。
- 当日の基本フロー、必要物、運搬、設営、撤収を運営モデルv0として整理する。

### 含めないこと

- 実施場所、日程、頻度、規模、参加人数の確定。

- 保険、事故対応、責任分界、現金管理ルールの確定。
- 実証実験の実行、施設への連絡、外部説明文の送信。
